<h2 style="color:#0C9;">9. Palindrome Number</h2>

Given an integer `x`, return `true` _if_ `x` _is a_ _**palindrome**_ _, and_ `false` _otherwise_.

**EXAMPLE**
>**Input**:  x = 121
**Output**: true

<p style="color:#007;">
<b>Explanation</b><br>
121 reads as 121 from left to right and from right to left.
</p>

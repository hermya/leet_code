<h2 style="color:#0C9;">1137. N-th Tribonacci Number</h2>

The Tribonacci sequence Tn is defined as follows: 

T~0~ = 0, T~1~ = 1, T~2~ = 1, and T~n+3~ = T~n~ + T~n+1~ + T~n+2~ for n >= 0.

Given `n`, return the value of Tn.

**EXAMPLE**
>**Input**: n = 4
**Output**: 4

<p style="color:#007;">
<b>Explanation</b><br>
T_3 = 0 + 1 + 1 = 2<br>
T_4 = 1 + 1 + 2 = 4<br>
</p>

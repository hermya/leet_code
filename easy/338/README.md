<h2 style="color:#0C9;">338. Counting Bits</h2>

Given an integer `n`, return *an array* `ans` *of length* `n + 1` *such that for each* `i` `(0 <= i <= n)`, `ans[i]` *is the **number of** `1`**'s** *in the binary representation of* `i`.

**EXAMPLE**
>**Input**: n = 5
**Output**: [0,1,1,2,1,2]

<p style="color:#007;">
<b>Explanation</b><br>
0 --> 0<br>
1 --> 1<br>
2 --> 10<br>
3 --> 11<br>
4 --> 100<br>
5 --> 101
</p>

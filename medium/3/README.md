<h2 style="color:#F90;">3. Longest Substring Without Repeating Characters</h2>

Given a string s, find the length of the longest substring without repeating characters.

**EXAMPLE**
>**Input**: s = "abcabcbb"
**Output**: 3


<p style="color:#007;">
<b>Explanation</b><br>
The answer is "abc", with the length of 3.

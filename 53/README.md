<h2 style="color:#F90;">53. Maximum Subarray</h2>

Given an integer array nums, find the subarray with the largest sum, and return its sum.


**EXAMPLE**
>**Input**: nums = [-2,1,-3,4,-1,2,1,-5,4]
**Output**: 6


<p style="color:#007;">
<b>Explanation</b><br>
The subarray [4,-1,2,1] has the largest sum 6<br> 
</p>
